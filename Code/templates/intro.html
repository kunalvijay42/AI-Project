Hand Gesture Recognition
About the System


Hand Gesture recognition system provides us an user friendly way of interaction with the computer which is more familiar to the human beings. In our project, the hand region is extracted from the background by Region of intrest. Then, we will be predicting the labels based on the CNN trained model weights of hand gestures using that predicted labels we apply if conditions to control some of the actions like reshaping , blur, flip of the given image.
